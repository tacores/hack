## 形式別のRCE危険性

| ファイル形式 | シリアライズ方式 | 主なリスク | 補足 |
|--------------|------------------|------------|------|
| .pkl / .pt / .bin | Python pickle | 読み込み時に任意コード実行 | ファイルをロードした瞬間に攻撃成立の可能性あり |
| .safetensors | 独自（生の重みのみ） | 低リスク（コード実行なし） | 安全性を重視した形式 |
| .h5 | Keras独自形式 | モデル構造に埋め込まれたコード実行の可能性 | pickleではないが完全に安全ではない |
| .gguf | 独自形式 | 重みレベルの攻撃・バックドア | シリアライズ起因のRCEはなし |

safetensors拡張子を偽装する場合がある点に注意 -> https://nvd.nist.gov/vuln/detail/CVE-2023-6730

### 無害な形式への変換

safetensorsに変換する例。weights_onlyを指定している分、全部ロードするよりは安全だが、pklをロードしている以上、絶対に安全かは疑問がある。

VMのような隔離された環境でsafetensorsとして保存、運用環境ではsafetensorsをロードとすればほぼ安全になると考えられる。

```sh
import torch
from safetensors.torch import save_file, load_file

# Step 1: Load the existing pickle model safely
# (weights_only=True restricts what pickle can do; explained in Defence 2 below)
model_weights = torch.load("model.pkl", weights_only=True)

# Step 2: Save as SafeTensors
save_file(model_weights, "model.safetensors")

# Step 3: Load the SafeTensors model (always safe)
safe_weights = load_file("model.safetensors")
```

## Keras Lambdaレイヤー

開発者が任意のPython関数をモデルに埋め込むことができるレイヤー。本来はトレーニング中のカスタム操作を目的としているが、攻撃者はこれを悪意のあるコードを隠すために悪用する可能性がある。

modelscan ではMIDIUMレベルとして表示される。具体的なコードが表示されない。

Lambdaレイヤーのコードを表示するスクリプト。

```python
import h5py
import json
import marshal
import types
import dis

model_path = 'your_model.h5'

with h5py.File(model_path, 'r') as f:
    # モデルの構成（JSON）を取得
    model_config = f.attrs.get('model_config')
    config = json.loads(model_config)

    for layer in config['config']['layers']:
        if layer['class_name'] == 'Lambda':
            print(f"--- Lambda Layer: {layer['name']} ---")
            
            # 関数情報を取得
            func_config = layer['config']['function']
            # Python 3系では通常ここに含まれるバイトコードを復元
            # 形式はKerasのバージョンに依存
            try:
                # marshalでシリアライズされたコードをロード
                # 実行環境とモデル作成時のPythonバージョンが異なると失敗する場合がある
                code_raw = eval(func_config[0]) 
                code = marshal.loads(code_raw)
                
                print("Disassembled Code:")
                dis.dis(code) # バイトコードを人間が読める形式に逆アセンブル
            except Exception as e:
                print(f"Could not disassemble: {e}")
```
