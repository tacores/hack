## バージョンピン止め

```sh
# BAD: allows any version
numpy
requests

# BETTER: pins major.minor but allows patches
numpy>=1.24,<1.25
requests>=2.31,<2.32

# BEST: pins exact version
numpy==1.24.3
requests==2.31.0
```

## ロックファイル

```sh
# hash 表記付きの requirements を出力する
pip-compile --generate-hashes
```

```sh
# pyproject.toml を入力として poetry.lock を出力
poetry lock

# インストール時に poetry.lock に厳格に従う（再解決しない）
poetry install
```

## 脆弱性スキャン

```sh
pip-audit -r /opt/supply-chain/project/requirements.txt
```

## プライベートパッケージインデックス

自組織でホストするインデックスを指定

```sh
# ~/.pip/pip.conf
[global]
index-url = https://your-private-pypi.company.com/simple/
extra-index-url = https://pypi.org/simple/
```

## SBOM (ソフトウェア部品表)

| 形式 | 管理元 | 強み |
|------|--------|------|
| SPDX | Linux Foundation | ライセンス遵守に重点を置き、ISO規格（ISO/IEC 5962:2021）に準拠 |
| CycloneDX | OWASP | セキュリティ重視、脆弱性データを含む、軽量 |


### SBOMを生成

syft -> https://github.com/anchore/syft

```sh
# CycloneDX JSON形式 で生成
syft /opt/supply-chain/project/ --exclude './venv/**' -o cyclonedx-json > /tmp/sbom.json
```

```sh
# 内容を改めて確認
syft /opt/supply-chain/project/ --exclude './venv/**' -o table

cat /tmp/sbom.json | python3 -m json.tool | less
```
