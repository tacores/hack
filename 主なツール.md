## Juice Shop

- kali 内ではなく Ubuntu 等の対向に入れても良い
- Node.js 前提
- port:3000

`git clone https://github.com/juice-shop/juice-shop.git --depth 1`

`cd juice-shop; npm install`

`npm start`
