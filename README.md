# hack

学習したサイバーセキュリティ技術の知識をため込むリポジトリです。

## ウイルス検知に関する注意

マルウェアやクラッキングツール等の技術情報を扱っているため、クローンすると Windows Defender 等がマルウェアとして検知してアラートが発報する可能性があります。そのため会社のPCなどにクローンするのはお勧めしません。

ただのMDファイル（テキストファイル）なので実際の危険はありません。

## ディレクトリ構造

CTF：Capture The Flag の記録

セキュリティアナリスト：SOC目線。フォレンジック技術など。

セキュリティエンジニアリング：防御システムを構築する役割の目線。

Windows：Windows 特有のハッキング技術

その他（大半）：全般的な攻撃視点の知識。整理したいという気持ちはあります。
