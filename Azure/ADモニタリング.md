## 認証イベント

### Kerberos

Kerberos認証時のSecurityイベント

| ステップ | 何が起きたか                           | イベントID | ログ記録場所               |
|----------|----------------------------------------------|------------|----------------------------|
| 1        | ユーザーがTGTをリクエストする                | 4768       | ドメインコントローラ       |
| 2        | ユーザーがTGSをリクエストする                | 4769       | ドメインコントローラ       |
| 3        | ユーザーがターゲット上でセッションを作成する | 4624       | ターゲットサーバー         |
| -        | 認証失敗 | 4771       | ドメインコントローラ        |


```
index=* EventCode=4768
| table _time, Account_Name, Client_Address, Ticket_Encryption_Type
```

#### チケット暗号タイプ

4768、4769に含まれる暗号タイプについて。  
0x12は最新でAES-256、0x17はレガシーなRC4暗号。

```
Ticket Encryption Type: 0x12
```

### NTLM

Kerberosが利用できない場合に使用されるレガシー認証プロトコル

| ステップ | 何が起きたか                                   | イベントID | ログ記録場所         |
|----------|------------------------------------------------------|------------|----------------------|
| 1        | ターゲットサーバーはDCに資格情報の検証を依頼する     | 4776       | ドメインコントローラ |
| 2        | ターゲットにセッションが作成されました               | 4624       | ターゲットサーバー   |

```
index=* EventCode=4776
| table _time, Logon_Account, Source_Workstation
```

```
index=* EventCode=4624 Account_Name=michelle.smith Authentication_Package=NTLM
| table _time host user Workstation_Name Source_Network_Address Authentication_Package
```

## アカウント、グループ、リソースアクセスイベント

### アカウント

| イベントID | 何が起きたか                           |
|------------|----------------------------------------|
| 4720       | アカウントを作成しました               |
| 4722       | アカウントが有効になりました           |
| 4724       | パスワードのリセットを試みました       |
| 4725       | アカウントが無効になりました           |
| 4740       | アカウントがロックアウトされました     |

作成されたアカウント、作成した人

```
index=* EventCode=4720
| table _time, SAM_Account_Name, Subject_Account_Name
```

### グループ

| イベントID | 何が起きたか                                                   | グループの範囲                                   |
|------------|----------------------------------------------------------------|--------------------------------------------------|
| 4728       | グローバル セキュリティ グループにメンバーが追加されました     | ドメイン全体                                     |
| 4732       | ローカル セキュリティ グループにメンバーが追加されました       | マシンレベル（DC上ではドメインローカル）         |
| 4756       | ユニバーサル セキュリティ グループにメンバーが追加されました   | フォレスト全体                                   |

```
index=* (EventCode=4728 OR EventCode=4732 OR EventCode=4756)
| table _time, Member_Account_Name, Group_Name, Subject_Account_Name
```

### ディレクトリサービスイベント

| イベントID | 何が起きたか                           |
|------------|----------------------------------------|
| 5136       | Active Directoryオブジェクトに対する属性レベルの変更               |

#### GPOの変更

```
index=* EventCode=5136 Class="groupPolicyContainer"
| table _time, Subject_Account_Name, DN, LDAP_Display_Name, Value
| sort - _time
```

#### ログオンイベント

| イベントID | 何が起きたか                           |
|------------|----------------------------------------|
| 4624       | ログオン成功               |
| 4625       | ログオン失敗               |

ログオンタイプ

| タイプ | 意味                     | 例                                                         |
|--------|--------------------------|------------------------------------------------------------|
| 2      | 相互（インタラクティブ） | キーボード操作、物理コンソールログオン                    |
| 3      | ネットワーク             | ファイル共有アクセス、WMIクエリ、リモート管理             |
| 4      | バッチ                   | ユーザーアカウントで実行されるスケジュールタスク          |
| 5      | サービス                 | サービスアカウントで起動するWindowsサービス               |
| 7      | ロック解除               | 以前にロックされたワークステーションのロック解除          |
| 10     | リモートインタラクティブ | RDPセッション                                              |

```
index=* EventCode=4624
| stats count by Logon_Type
| sort -count
```

## テクニック

### スタックカウント（ロングテール分析）

稀なアクティビティに注目する例

```
index=* EventCode=4769 NOT Account_Name="*$*"
| stats count by Account_Name
| sort -count
```
