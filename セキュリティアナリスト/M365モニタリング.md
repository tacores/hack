## Identity Provider (IdP)

Microsoft Entra IDはクラウドベースのIDプロバイダーの一例。（Azure Active Directory は古い呼び方）  
他には、Twitter、Google、Amazon、LinkedIn、Appleなどが挙げられる。

例：Googleアカウントの認証情報を使用してSpotifyにログインできる。この場合、GoogleログインがIdP、Spotifyがサービスプロバイダ（SP）となる。

Identity は、人間、ワークロード、デバイスの3カテゴリに分類される。

## 侵害のリスクを高める要素

- 多要素認証(MFA)の欠如
- 過度に許可されたアクセスポリシー
- 過剰な管理者権限
- 弱いパスワードポリシー
- 無効な認証リスクポリシー
- 不十分なログ記録と監視

## ログモニタリング

エラーコードの意味を確認する便利ツール  
https://login.microsoftonline.com/error

### Entra ID サインインログ

全てのサインインイベント

```
index=* sourcetype="azure:aad:signin"
```

失敗したサインインイベント

```
index=* sourcetype="azure:aad:signin" "status.errorCode"!=0
| stats count as event_count values(ipAddress) as ip_addresses
 values(appDisplayName) as applications values(status.errorCode) as errorCodes  by userPrincipalName
| sort - event_count
| table applications, userPrincipalName, ip_addresses, errorCodes, event_count
```

エラーコードの例

- 50126: ユーザー名またはパスワードが無効です
- 50053: 試行回数が多すぎるためアカウントがロックされました
- 50074: MFA が必要ですが提供されていません
- 50055: パスワードの有効期限が切れています

特定のIPアドレスから成功したサインインイベント

```
index=* sourcetype="azure:aad:signin" "status.errorCode"=0 ipAddress="<ADD-IP-HERE>"
| stats values(ipAddress) as ip_addresses values(appDisplayName) as applications  by userPrincipalName
| table applications, userPrincipalName, ip_addresses
```

### 監査ログ

監査ログとは、Entra ID環境内で行われた管理アクションと変更。

全ての監査ログ

```
index=* sourcetype="azure:aad:audit"
```

特定のユーザーを対象とした変更

```
index=* sourcetype="azure:aad:audit" targetResources{}.userPrincipalName="<ADD-USER-EMAIL>" 
| eval initiator=coalesce('initiatedBy.user.userPrincipalName', 'initiatedBy.app.displayName')
| sort - _time
| table _time, initiator, activityDisplayName, result, targetResources{}.userPrincipalName
```

ユーザーが実行した変更

```
index=* sourcetype="azure:aad:audit" initiatedBy.user.userPrincipalName="<ADD-USER-EMAIL>" 
| sort - _time
| table _time, initiatedBy.user.userPrincipalName, activityDisplayName, result, targetResources{}.userPrincipalName
```

### M365 監査ログ

全ての M365監査ログ

```
index=* sourcetype="o365:management:activity"
```

ユーザーが実行したアクション

```
index=* sourcetype="o365:management:activity" UserId="<ADD-USER-EMAIL>"
| sort - _time
| eval sourceIP=coalesce('ClientIP', 'ClientIPAddress')
| table _time, Operation, UserId, sourceIP, Workload, ObjectId
```
